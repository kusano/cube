Linear FMC Counter

ルービックキューブ関連 > Linear FMC Counter

スマートキューブを使ってcsTimerで記録した手順を、3x3x3 Linear Fewest Moves（Linear FMC）のルールに従って数えます。

詳細

スマートキューブをcsTimerに接続し、キューブを解いてください。 解いた後、画面右端の手数をクリックして仮想キューブを開き、（share linkではなく）⏯Algのリンクをコピーして貼り付けてください。 L D R' ... のような手順を貼り付けることもできます。

Linear FMCのルールでは、R R R → R' のように、同じ面を回す連続した手順を一つにまとめます。 まとめた結果が0となった場合は、R R' → R0 のように残します。 この動きは持ち替えと同様に手数としてカウントはされませんが、この R0 などを跨いだ動きを一つにまとめることはしません。 F R R' F は、F R0 F になり、2手としてカウントします。
