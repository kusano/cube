OLL

理論

エッジは、U面がU面色かどうかで側面がU面色でないかどうかが分かる。 コーナーは、1個は3面が見えている。 2個は2面が見えているので、U面色がどの面にあるのかが分かる。 残りの1個のコーナーは、コーナーの回転の合計が3の倍数になるというキューブの性質から分かる。 コーナーは、次のいずれかの配置を回転させたものになる。

普通に（3側面以上を見て）OLLの判断ができている時点で2側面でもだいたいは判断でき、判断に使っている部分が見えない面にある場合を追加で覚えるだけで良いはず。

PLL

用語

このページでは次のように定める。

バー
ある側面の3個のピースが全て同色。
ヘッドライト
ある側面の2個のコーナーが同色で、エッジの色が異なる。
ペア
ある側面のエッジと一方のコーナーが同色で、他方のコーナーの色が異なる。
ブックエンド
両端のピースが同色。

ヘッドライトとペアについて、ペッドライトやペアの色と残りのピースの色が隣接色であれば、そのヘッドライトやペアも隣接色と呼ぶことにする。 対面色の場合も同様。

理論

コーナーについて、コーナーは次の完成状態、対角交換、隣接交換のいずれかになる。 どの方向から見ても、2側面を見れば、どの状態かが分かる。

完成状態は、どの側面もコーナーの色が揃っている。 すなわち、2側面のどちらもバーかヘッドライトになる。

対角交換は、どの側面もコーナーの色は対面色の組み合わせになっている。

隣接交換は、どの方向から見てもコーナーが隣接色の組み合わせになっている側面がある。 もう一方の側面は、色が揃っているか、対面色の組み合わせになっているかのどちらか。 対面色の組み合わせになっている方向から見たときにはブックエンドがある。 また、色が揃っている側面は、対面色の組み合わせの側面の対面にある。

Eパームのような隣接交換2回は、対角交換から U もしくは U' した状態である。 隣接交換から U もしくは U' するとAパームのような3点交換になる。

コーナーの状態について、どのPLLがありえるかを分類すると次の表になる。 この表を意識しておくと、判断の内容が頭に入ってきやすいかもしれない。

コーナー PLL
完成状態H, U, Z
対角交換E, N, V, Y
隣接交換A, F, G, J, R, T

エッジは、2個のエッジが見え、2個のエッジは見えない。 コーナーとエッジの置換の偶奇は一致するというキューブの性質から、見えない2個のエッジの配置が推測できる。

これによって、見えない側面の配色を全て推測することはできる。 できるが、時間が掛かるので、実際にはバーやヘッドライトの有無などから判断する。

ペアが1個とブックエンドがある

  • ペアが内側にある
    • この時点でGパームであることは確定する。
    • ペアが隣接色
      • → Ga
      • → Gc
    • ペアが対面色
      • → Gb
      • → Gd
  • ペアが外側にある
    • ペアが隣接色
      • ペアがある側面のペアになっていないコーナーと、ペアではない側面のエッジが同色
        • → Ra
        • → Rb
        • ヘッドライトはペアに隣接している。
      • ペアがある側面のペアになっていないコーナーと、ペアではない側面のエッジが異色
        • → T
      • 色が多くて側面が複雑に見えるほうが、簡単なTパームになる。
    • ペアが対面色
      • ペアがある側面のペアになっていないコーナーと、ペアではない側面のエッジが同色
        • → Gb
        • → Gd
      • ペアがある側面のペアになっていないコーナーと、ペアではない側面のエッジが異色
        • → Aa
        • → Ab
      • 色が多くて側面が複雑に見えるほうが、簡単なAパームになる。

ペアが1個ある

  • ペアが内側にある
    • → Y
  • ペアが外側にある
    • → V

ブックエンドがある

  • 中央4個に2色が2個ずつある
    • → F
    • 2個のコーナーが対面色の側面の対面にバーがある。
  • 中央4個に2個ある色がブックエンドと隣接色
    • → Ra
    • → Rb
    • 2個のコーナーが対面色の側面の対面にヘッドライトがある。
  • 中央4個に2個ある色がブックエンドと対面色
    • → Ga
    • → Gc
    • 2個のコーナーが対面色の側面の対面にヘッドライトある。

バー、ヘッドライト、ペア、ブックエンドが無い

  • 中央4個の色が交互になっている
    • → V
    • 反対側の中央4個がペアになっている。
  • 外側4個の色が交互になっている
    • → Y
    • 反対側の外側4個がペアになっている。
  • 5個の色が回文のようになっている
    • → E
    • Eパームはどこから見てもこの状態であり、条件分岐のここにしか出てこない。
